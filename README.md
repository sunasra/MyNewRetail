To Run Project:
*npm install*
*npm start*

To make prod buid
*npm run prod*

To Run test cases:
Install nightwatch globally
1. *npm install -g nightwatch*
2. *npm start*
3. *cd tests*
4. *java -jar selenium-server-standalone-3.4.0.jar*  //this will run selenium server
5. *npm run test*

*npm run test*
